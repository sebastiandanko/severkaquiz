# Informace o placeholder obrázcích
Aplikace očekává následující strukturu obrázků zvířat:

## Adresář zvířat (images/animals/)
- sykora_konadra.jpg - Sýkora koňadra image
- vrabec_domaci.jpg - Vrabec domácí image  
- zajic_polni.jpg - Zajíc polní image
- mlok_skvrnity.jpg - Mlok skvrnitý image
- rys_ostrovid.jpg - Rys ostrovid image
- vrana.jpg - Vrána image
- havran.jpg - Havran image
- srnec.jpg - Srnec image
- jelen.jpg - Jelen image

## Adresář stop (images/tracks/)
- sykora_track.png - Sýkora track
- vrabec_track.png - Vrabec track
- zajic_track.png - Zajíc track
- mlok_track.png - Mlok track
- rys_track.png - Rys track
- srnci_track.jpg - Srnec track

## Jak přidat reálné obrázky:
1. Umísti fotografie zvířat do images/animals/ directory
2. Umísti fotografie stop do images/tracks/ directory
3. Ujisti se, že názvy souborů odpovídají těm v quiz-data.js
4. Podporované formáty: JPG, PNG, GIF

## Pro testování bez reálných obrázků:
Použijte zástupné obrázky nebo aktualizuj quiz-data.js tak, aby používal externí adresy URL obrázků
